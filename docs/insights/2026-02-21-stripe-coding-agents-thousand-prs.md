# Stripe's Coding Agents Ship 1,000+ PRs Weekly

**Key finding:** Stripe's homegrown AI coding agents — internally called "Minions" — are writing and merging more than 1,000 pull requests per week, with humans reviewing but not writing the code.

**Why it matters:** This isn't a benchmark or research demo. Stripe is one of the world's most scrutinised engineering organisations, operating under tight security and compliance requirements. The fact that one-shot, end-to-end AI agents have reached this throughput in that environment means the capability is real, mature, and ready for broad adoption. The second instalment of their engineering blog deepens the technical picture, showing how these agents handle complex, multi-step coding tasks without human handholding at each step. When Stripe publishes production numbers, the industry listens.

**Implication / so what:** Engineering velocity is being permanently repriced. Teams that build agent-augmented development pipelines now will compound a throughput advantage that is very hard for slower movers to close. For investors, any business where skilled technical labour is the bottleneck — software, infrastructure, fintech — is a candidate for step-change cost and speed improvements. The question is no longer "can AI code?" It's "how fast are you integrating it?"
